TssGrid — demo

軽量・ノービルド・日本語IME堅牢な編集グリッド。
操作: 矢印=移動 / Shift+矢印・ドラッグ=範囲 / 行・列ヘッダのクリック=行/列選択（角で全選択） / 列ヘッダ右端・行ヘッダ下端をドラッグ=幅/高さ変更（線でプレビュー→離して確定） / F2・ダブルクリック・文字=編集 / Enter=下 / Tab=右 / Delete=範囲クリア / Ctrl+C/V/X=コピペ(Excel相互) / Ctrl+Z・Y=Undo/Redo / 右クリック=行/列の挿入・削除メニュー / フィルハンドル右下ドラッグ=コピー（数値2つ=連番 / 単一+Ctrl=+1 / Alt=2D矩形）。

1) 単体（HistoryManager を渡さない＝自前で生成）

1.5) セルタイプ + バリデータ（dropdown / checkbox / date / 入力検証）

状態列=ドロップダウン / 完了列=チェックボックス（Space・クリックでトグル。ヘッダのチェックで全選択/全解除＝部分選択は indeterminate） / 期限列=日付（不正日付は拒否） / 期限列=日付（20260630 や 2026/6/30 と打てる→保存は ISO・表示は yyyy/mm/dd） / 時刻列=時刻（12時間制 AM/PM 表示。1:30 PM も 13:30 も入力可、下のボタンで 24h と切替） / 数量列=validator（数値のみ）。貼り付け・フィルでも同じ検証が効きます。 このデモは invalidMode:'keep'＝検証 NG は値を残して赤く表示（直せば赤が消える）。

1.6) CellFormat（表示書式：数値カンマ/小数/通貨・日付パターン）

保存値は正規のまま（getData は素の値）、表示だけ整形します。
単価/金額列=¥ + カンマ（190200 と打てる→¥190,200 表示。編集時は素の数値） / 達成率列=小数1桁 + % / 納品日列=format:'yyyy年m月d日'（20260630 でも入力可）。 数値列は右寄せ。非数値・不正日付は弾きます。
このデモは width:560 で枠を制約＝はみ出す列はテーブル自身の横スクロールバーで送れます（未指定だと枠が内容幅まで広がりページ側がスクロール）。

1.7) 列固定（frozenCols）＋縦横スクロール

左の 2列（ID・担当）を固定＋ヘッダ行も固定。width / height で枠を制約して縦横スクロールします。 「状態」列は resizable:false で幅ロック（その列の右端にリサイズグリップが出ません）。

1.8) 読み取り専用（readOnly）

ID 列と小計 列は readOnly:true（淡色＝編集不可。クリック・F2・貼り付け・フィル・Delete のどれでも変更を拒否し元の値に戻します）。
数量 / 単価は編集可。状態列は 関数 readOnly（「完了」の行だけロック）。読み取り専用セルへの変更は onReadOnly で通知されます。

1.9) 右クリックメニュー（行/列の挿入・削除）

セルやヘッダを右クリックすると、行/列の挿入・削除＋コピー/切り取り/貼り付け/クリアのメニューが出ます（複数行/列を選択してまとめて削除も可）。挿入・削除は Undo/Redo に1操作で積まれます。
各操作は allowInsertRows / allowDeleteRows / … で個別に ON/OFF 可。下のボタンで「行の追加/削除」を切替えて右クリックで確認できます。contextMenu に配列を渡せば項目の取捨選択＋独自項目・サブメニューも足せます（README 参照）。

1.10) データの持ち方（配列 / オブジェクト配列）

columns[c].data='キー' で列に名前を紐付けると、data にオブジェクト配列を渡せます（従来の配列形式も自動判別でそのまま使えます）。
grid.getValue(r,'キー') で名前アクセス、grid.getRows() でオブジェクト配列として取り出し。下のグリッドは id という画面に出さないフィールドを持っていて、編集・行挿入・削除をしても getRows() で保持されます。
「品名列を隠す/表示」は toggleColumn(1)（列の表示/非表示をプログラムから制御）。隠してもデータは保持され、getRows() にも出ます。 



1.11) 入力フォーム向けオプション（placeholder / minSpareRows / autoWrapRow / stretchH）



placeholder＝空セルに薄いヒント / minSpareRows:1＝末尾に常に空行（下へ打ち足せる） / autoWrapRow＝Tab で行末から次行頭へ折り返し / stretchH:'last'＝最終列を枠幅まで伸ばす。

試しに一番下の空行に入力 → さらに空行が増えます。Tab で右端まで行くと次の行の先頭へ移ります。






1.12) 整列 / セル読取専用（実行時・プログラム制御）



範囲を選んでボタン操作。setAlignment('left'|'center'|'right') で整列（数値列の既定右寄せも上書き可）、setCellReadOnly(true/false) でセル単位の読取専用を実行時に切替。どちらも列の挿入・削除でセルにちゃんと追従します。




  
  
  
  
  
  






1.13) 列幅オートフィット / autofill拡張 / カスタムエディタ



列幅オートフィット: ボタン、または列ヘッダ右端をダブルクリックで内容に合わせて自動調整。

autofill拡張: このグリッドは fillHandle:{autoInsertRow:true}＝フィルハンドルをテーブル下端より下までドラッグすると行が自動追加されます。

カスタムエディタ: 「メモ」列は columns[c].editor で textarea（複数行）編集（Ctrl+Enter または外側クリックで確定 / Esc 取消）。確定は検証・Undo の関所を通ります。




  






1.14) ショートカット / プラグイン / 破棄(destroy)



Ctrl+D＝選択行を下に複製（カスタムショートカット）。プラグインで編集のたびに件数を表示。破棄ボタンで destroy()＝リスナ/Observer を全解除して DOM ごと撤去（SPA のリーク防止）。
