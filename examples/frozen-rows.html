TssGrid 例: 行固定 frozenRows

上から N 行を縦スクロールしても固定します（ヘッダ行の sticky と同じ仕組みの行版）。列固定 frozenCols と併用すると四隅が固定されます。
このデモ：frozenRows: 1（合計行を上に固定）＋ frozenCols: 1（品番を左に固定）。縦にも横にもスクロールして確認してください。

差し込み

new TssGrid(host, {
  height: 280,            // 縦スクロールを出す（sticky の効く枠）
  frozenRows: 1,          // 上から1行を固定（実行時 grid.freezeRows(n) でも）
  frozenCols: 1,          // 左から1列を固定（併用で四隅固定）
  data: rows,
});
