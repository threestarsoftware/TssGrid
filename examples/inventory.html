TssGrid 例: 簡単な在庫管理 機能てんこ盛り

TssGrid 1本（＋カレンダープラグイン）で作った在庫管理ミニアプリ。これくらいの業務画面がノービルドで組めます（項目を差し替えれば勤怠・発注・工程にも転用可）。
使っている機能：ドロップダウン(カテゴリ) / 数値書式(数量・単価・金額) / 計算列(金額＝数量×単価, 読み取り専用) / チェックボックス＋ヘッダ全選択(在庫) / 休日カレンダー(入荷日) / 入力検証(品番) / ヘッダクリックで並べ替え / 右クリックで行挿入削除・全チェック / 末尾に常に空行 / Excel相互コピペ・Undo / getRows() で保存。
