TssGrid 例: セルに画像 / SVG / HTML html:true

列に html:true を付けると、その列は表示だけ生HTMLになります（値 or format の戻りを innerHTML）。値はテキスト保存のままなので getData()・コピペ・CSV は無傷——見た目だけHTML化します。画像・SVGアイコン・状態バッジ・進捗バー・評価★などに。
⚠️ エスケープを外す＝XSSの責任は format/値の提供側。外部由来の文字列をそのまま埋めないこと。表示用途なので各列は readOnly 推奨。

デモ（表示専用）

差し込み（要点）

columns: [
  { data:'名前' },
  { data:'状態',   html:true, readOnly:true, format: badge },   // SVGドット＋ラベル
  { data:'達成率', html:true, readOnly:true, format: bar },     // 進捗バー（div）
  { data:'評価',   html:true, readOnly:true, format: stars },   // ★レーティング
  { data:'アイコン', html:true, readOnly:true },                 // 値に inline SVG をそのまま
]
