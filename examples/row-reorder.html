TssGrid 例: 行のドラッグ移動 rowReorder

行ヘッダ左のつまみ「⠿」をドラッグすると行を並べ替えられます。ドロップ位置に緑のガイド線が出ます。
複数行の移動も可：行ヘッダで範囲選択（クリック→Shift＋クリック、または行ヘッダをドラッグ）してから、選択内のつまみを掴むとまとめて移動します。
データ実体ごと移動するので、セルの値・整列・読み取り専用・隠しフィールドも追従します（並べ替えと同じ仕組み）。行番号は自動で振り直し。

デモ（⠿ をドラッグ）

 

差し込み（抜粋）

new TssGrid(host, {
  rowReorder: true,                       // 行ヘッダに「つまみ」を出してドラッグ移動
  onAfterRowMove: (from, to) => { ... },  // 確定後に呼ばれる（任意）
  data: rows,
});
// プログラムからは grid.moveRow(from, to) でも移動可
